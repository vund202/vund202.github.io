Adding More Slot Types for Better Responses

To enhance your chatbot’s response handling, follow the steps below to create additional slot types and integrate them into the conversation flow.

1. Create More Slot Types

Now, you need to create 3 more slot types: ExtraShot, Caf, and DrinkType, just like the steps covered in Handling Incorrect Inputs in Your Chatbot.

2. Add Slots to the Intent

Once the slot types are created, add them into the Slots section of the relevant intent.

3. Conditional Branching for Slot Prompts

At the Prompt for the DrinkType slot, select Advanced options, scroll down, and choose Branch 1 under Conditional Branching.

  1. Set up the condition:
    • Type {DrinkType} = "coffee".
    • Choose Set Value and in Next step in conversation, select Elicit a slot. Then choose Extra Shot as the slot.

  1. Add a second branch for Cappuccino:
    • Type {DrinkType} = "cappuccino", and set the slot to Extra Shot.

  1. Add a third branch for Tea:
    • Type {DrinkType} = "tea", and set the slot to Caf (caffeine level).

4. Update and Confirm Slots

Once you’ve added all conditional branches, make sure to update the slots to match the appropriate inputs and outputs.

Next, add parameters in the Confirmation section to ensure that the bot confirms the chosen drink and its details before finalizing the order.

5. Build and Test the Chatbot

After completing the slot and confirmation configurations, click Build to apply your changes.

Now you can test the chatbot to see the result.

With these additional slot types and the conditional branching, your chatbot is now more dynamic and responsive. It can handle a variety of drink options, prompting users for further details based on their selections, ensuring a smooth and interactive user experience.
